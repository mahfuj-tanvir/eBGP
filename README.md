# Enterprise Multi-Site LAN Integration with Redundancy and Internet Connectivity
Description: Designed and implemented a redundant multi-branch enterprise network with dynamic routing, secure internet connectivity, and high availability using Cisco Packet Tracer.
Key Implementation:
•	Configured eBGP routing between ISP, CORE, and branch routers for scalable connectivity
•	Implemented HSRP for gateway redundancy and automatic failover
•	Deployed NAT/PAT on CORE routers for secure internet access
•	Managed IP addressing and inter-branch communication
•	Performed network verification and troubleshooting using Cisco diagnostic commands
Technologies & Tools: Cisco Routers, Switches, PCs, eBGP, HSRP, NAT/PAT, IPv4 Addressing, Cisco Packet Tracer
